retrotune

april 2024


retrotune is a web application that determines different "eras" in your life by analyzing your Instagram feed and curates Spotify playlists based off the vibe. this was for LA Hacks. the project uses Python, Reflex, Instagram API, Spotify API, and Google Gemini API. i set up and routed the backend, tuned the gemini AI model to analyze images for our purpose, connected the backend and frontend, built the spotify playlist creation feature, and created the frontend for two pages.

text to tracks

april 2024


text to tracks takes a link to an article, scrapes the text, and puts it in the TikTok Subway Surfers-style video format. this won an honorable mention at UCSD's Diamondhacks Hackathon. i routed the backend, connected the frontend and backend, and created the frontend pages. this project features Python, HTML/CSS, Flask, OpenCV, goose3, Vosk API, and various other Python libraries.
